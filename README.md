# About the Project

This project is a modern fashion package landing page designed for brands that want to advertise premium product bundles.

Instead of displaying products individually, the landing page groups multiple products into attractive packages that encourage customers to purchase bundled offers.

The design follows a clean Apple-inspired aesthetic with generous whitespace, premium typography, soft shadows, and elegant animations that improve user engagement and conversions.

# How It Works

When a visitor opens the website:

1. They are greeted by a premium hero section introducing the fashion packages.
2. The call-to-action button smoothly scrolls them to the pricing section.
3. Visitors explore the products included in each package.
4. They compare the available package options.
5. Clicking a package button is currently connected to a placeholder action, which can easily be replaced with:
   - Paystack Checkout
   - Flutterwave
   - Stripe
   - Custom checkout flow
6. The trust footer reassures customers that secure online payments are supported.
